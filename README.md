# Interior Worlds — Interior Design Business Website

A professional single-page website for an interior design business serving clients across India and the UAE.

## What it does

- Showcases interior design services and philosophy
- Embeds YouTube portfolio videos with autoplay (muted)
- Prominent YouTube subscribe button to grow the channel
- Two clearly differentiated WhatsApp contact buttons (India 🇮🇳 and UAE 🇦🇪)
- Links to YouTube, Instagram, and Facebook social profiles
- Enquiry contact form for potential clients
- Mobile-responsive design with professional teal/cyan/emerald colour palette
- Background images sourced from Unsplash (copyright-free)

## Tech stack

- **Framework**: TanStack Start (React 19)
- **Styling**: Tailwind CSS v4 + custom CSS
- **Fonts**: Cormorant Garamond (display) + DM Sans (body) via Google Fonts
- **Routing**: TanStack Router
- **Deployment**: Netlify

## Running locally

```bash
npm install
npm run dev
```

The site runs on `http://localhost:3000` by default.

> **Note**: Replace the placeholder WhatsApp numbers in `src/routes/index.tsx` with the real India and UAE numbers before publishing.
